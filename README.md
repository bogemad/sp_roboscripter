# sp_roboscripter

## Script descriptions

### `sp_roboscripter_template_24-48-96.py`
This is the template for the GUI

### `sp_robotics_template_24-48-96.py`
This is the template for the OpenTrons script

### `sp_roboscripter_24-48-96.py`
This is the combined GUI/Opentrons script that will output custom OpenTrons scripts based on your sample details.

#### make `sp_roboscripter_24-48-96.py` using the make_robotscripter.py script:

    make_robotscripter.py sp_roboscripter_template_24-48-96.py sp_robotics_template_24-48-96.py sp_roboscripter_24-48-96.py
